=== INSTRUCOES DE INSTALACAO ===

1. Copie toda esta pasta para o computador destino
2. Execute teste.bat para verificar se o servico esta funcionando
3. Os logs serao gerados na pasta 'logs'
4. Em caso de problemas, verifique:
   - Se o arquivo config.json esta configurado corretamente
   - Se os certificados estao presentes na pasta certs
   - Se o banco de dados esta acessivel no caminho configurado
   - Se ha permissao de escrita na pasta logs
